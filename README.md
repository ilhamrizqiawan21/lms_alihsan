## 📖 About

LMS MTs. Al-Ihsan Batujajar is a modern Learning Management System designed to support school administration and digital learning activities.

The system provides complete academic management for administrators, principals, teachers, and students through a clean and scalable Laravel architecture.

# ⚡ Installation

```bash
git clone https://github.com/username/lms_alihsan

cd lms_alihsan

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate --seed

npm install

npm run build

php artisan serve
```
